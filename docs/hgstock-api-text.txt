Tài liệu về tích hợp tải lên HGStock – API mở

Par Anh Can Viet (Unlicensed)

Ngày 08 tháng 01 năm 2026

Xác thực

Tất cả các request REST Admin API đều yêu cầu token hợp lệ để có thể truy xuất thông tin.

Sử dụng token trong header Authorization trong tất cả các API requests.

curl -X GET https://stock-api.hgmedia.app/demo-api \
 -H 'Content-Type: application/json' \
 -H 'Authorization: Bearer {token}'

Với quy trình xác thực khách hàng 

curl -X POST {identity-server-url}/connect/token \
     -H "Content-Type: application/x-www-form-urlencoded" \
     -d "grant_type=client_credentials" \
     -d "client_id={client_id}" \
     -d "client_secret={client_secret}" \
     -d "scope={scope}"

client_idvà client_secret, scopephải được admin HGStock cung cấp trước.

 Với Implicit Flow

curl "{identity-server-url}/connect/authorize?client_id={client_id}&redirect_uri={redirect_uri}&response_type=token&scope={scope}"

client_secret không bắt buộc trong Implicit Flow.

 Thông số chung:

identity-server-url: URL của Identity Server, thay đổi tùy theo môi trường sản phẩm.

Môi trường Development: https://accounts.hgmedia.dev

Môi trường Production: https://accounts.hgmedia.app

Lỗi

Mã lỗi

400 Bad Request: data format không đúng định dạng.

401 Unauthorized: client không có thông tin xác thực hợp lệ

403 Forbidden: server từ chối phản hồi. Trường hợp chủ yếu là do ứng dụng sử dụng token vào các api không được cấp quyền.

404 Not Found: tài nguyên được yêu cầu không tìm thấy, tuy nhiên có thể tìm thấy trong tương lai.

429 Too Many Requests: client vượt quá ngưỡng giới hạn request

5xx Errors: lỗi hệ thống nội bộ.

Định dạng dữ liệu

Ngày giờ :

Tuân thủ theo chuẩn ISO-8601 với timezone UTC.

Tất cả các dữ liệu datetime trong request và response của REST API đều tuân thủ theo quy chuẩn này

Ví dụ:

Lời yêu cầu

{
  "resourceFile": {
    "createdDate": "2002-11-04T08:05:03Z"
  }
}

Phản ứng

{
  "resourceFile": {
    "updatedDate": "2002-11-04T08:05:03Z"
  }
}

API chứng khoán

Thông số chung:

domain_url: URL của Stock BE, thay đổi tùy theo môi trường sản phẩm.

Môi trường Development: https://stock-api.hgmedia.dev

Môi trường Production: https://stock-api.hgmedia.app

Tiêu đề

X-Time-Zone: Timezone của client, ví dụ: Asia/Bangkok

Authenticantion: Mã thông báo người mang
 Với client sử dụng Client Credentials Flow, userId được gửi qua query param, ví dụ:

curl -X GET https://stock-api.hgmedia.app/demo-api?userId=1123xxx \
 -H 'Content-Type: application/json' \
 -H 'Authorization: Bearer {token}'

POST /api/v1/external/get-presigned-url

Description: Tạo presigned url để upload tệp lên server lưu trữ

Lời yêu cầu

curl -X 'POST' \
  '{domain_url}/api/v1/external/get-presigned-url' \
  -H 'accept: */*' \
  -H 'X-Time-Zone: Asia/Bangkok' \
  -H 'Authorization: Bearer {token}' \
  -H 'Content-Type: application/json' \
  -d '{

      "partNumber": 0,

  private static async Task<string> ReadChunkAsync(Stream fileStream, long start, int size)
        {
            byte[] buffer = new byte[size];
            fileStream.Seek(start, SeekOrigin.Begin);
            int bytesRead = await fileStream.ReadAsync(buffer, 0, size);
            return Encoding.GetEncoding("ISO-8859-1").GetString(buffer, 0, bytesRead);
        }

        private static async Task<string> GetSHA256ToStrAsync(string str)
        {
            using (SHA256 sha256Hash = SHA256.Create())
            {
                byte[] data = Encoding.GetEncoding("ISO-8859-1").GetBytes(str);
                byte[] hashBytes = await Task.Run(() => sha256Hash.ComputeHash(data));
                StringBuilder hashHex = new StringBuilder(hashBytes.Length * 2);
                foreach (byte b in hashBytes)
                {
                    hashHex.Append(b.ToString("x2"));
                }
                return hashHex.ToString();
            }
        }
        public static async Task<string> GetHashCodeOfFileAsync(Stream fileStream, int chunkSize = 5 * 1024 * 1024)
        {
            const int numberOfChunks = 3;
            string concatenatedHash = string.Empty;
            long fileSize = fileStream.Length;

            if (fileSize > (chunkSize * 3))
            {
                for (int i = 0; i < numberOfChunks; i++)
                {
                    long startPosition = (i == 0) ? 0 : (i == 1 ? (fileSize / 2) - (chunkSize / 2) : fileSize - chunkSize);
                    string readChunks = await ReadChunkAsync(fileStream, startPosition, chunkSize);
                    string hash = await GetSHA256ToStrAsync(readChunks);
                    concatenatedHash += hash;
                }
            }
            else
            {
                int actualNumberOfChunks = Math.Min(numberOfChunks, (int)Math.Ceiling((double)fileSize / chunkSize));
                for (int i = 0; i < actualNumberOfChunks; i++)
                {
                    long startPosition = i * chunkSize;
                    string readChunks = await ReadChunkAsync(fileStream, startPosition, chunkSize);
                    string hash = await GetSHA256ToStrAsync(readChunks);
                    concatenatedHash += hash;
                }
            }

            return await GetSHA256ToStrAsync(concatenatedHash);
        }

export const readChunk = (file: any, start: number, size: number): Promise<string> => {
  return new Promise((resolve, reject) => {
    const reader = new FileReader();
    const blob = file.slice(start, start + size);
    reader.onload = (event) => {
      resolve(event.target.result as string);
    };
    reader.onerror = (error) => {
      reject(error);
    };
    reader.readAsBinaryString(blob);
  });
};

export const getSHA256ToStr = async (str: any): Promise<string> => {
  if (typeof window !== 'undefined' && window.crypto && window.crypto.subtle) {
    const encodeToLatin1 = (str) => {
      const bytes = new Uint8Array(str.length);
      for (let i = 0; i < str.length; ++i) {
        bytes[i] = str.charCodeAt(i) & 0xff;
      }
      return bytes;
    };

    const data = encodeToLatin1(str);
    const hashBuffer = await crypto.subtle.digest("SHA-256", data);
    const hashArray = Array.from(new Uint8Array(hashBuffer));
    const hashHex = hashArray.map((byte) => byte.toString(16).padStart(2, "0")).join("");
    return hashHex;
  }

  const hash = CryptoJS.SHA256(CryptoJS.enc.Latin1.parse(str));
  return hash.toString(CryptoJS.enc.Hex);
};

export const getHashCodeOfFile = async (file: any, chunkSize: number): Promise<string> => {
  const numberOfChunks = 3;
  let concatenatedHash = '';

  if (file.size > (chunkSize * 3)) {
    for (let i = 0; i < numberOfChunks; i++) {
      const startPosition = (i === 0) ? 0 : (i === 1 ? (file.size / 2) - (chunkSize / 2) : file.size - chunkSize);
      const readChunks = await readChunk(file, startPosition, chunkSize);
      const hash = await getSHA256ToStr(readChunks);
      concatenatedHash += hash;
    }
  } else {
    const actualNumberOfChunks = Math.min(numberOfChunks, Math.ceil(file.size / chunkSize));
    for (let i = 0; i < actualNumberOfChunks; i++) {
      const startPosition = i * chunkSize;
      const readChunks = await readChunk(file, startPosition, chunkSize);
      const hash = await getSHA256ToStr(readChunks);
      concatenatedHash += hash;
    }
  }
  return getSHA256ToStr(concatenatedHash);
};
